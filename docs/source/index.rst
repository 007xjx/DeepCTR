Welcome to DeepCTR's documentation!
===================================

DeepCTR is a easy-to-use,modular and extendible implemention of deep-learning based CTR models ,
including FNN,PNN,WDL,DeepFM,NFM,AFM,,AFM,DCN,DIN etc

You can find source code at https://github.com/shenweichen/DeepCTR
